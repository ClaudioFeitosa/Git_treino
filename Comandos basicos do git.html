comandos basicos do git

Git init

Este comando inicialisa seu reporsitorio, ele cria uma pasta git dentro da pasta do seu projeto Este comando você so vai utilizar uma vez por projeto não ha necessidade de fazer o git init mais de uma vez para o mesmo projeto


git add.

Este comando serve para adicionar os arquivos/modificações ele vai enviar os arquivos para Stage

git commit -m "mensagem explicativa"
git status

Este comando mostra o status do projeto, você pode usar ele a qq momento se usar logo apos o seu git add você vai ver a lista de arquivos que estão prontos para subir

Este comando serve para você colocar uma mensagem no pacote por exemplo indicar o que foi modificado neste pacote

git branch -M main
git remote add origen git@github.com:claudiofeitosa/Git_treino.git
git push -u origin main
